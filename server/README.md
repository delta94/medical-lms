## Installation

```bash
$ npm install
```

## Running the app
You must have the following environment variables set for the database.

DB_HOST

DB_PORT

DB_SCHEMA

DB_USER

DB_PASSWORD
```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
